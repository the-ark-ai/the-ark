# Contributing to The Ark AI

Thank you for your interest in contributing to The Ark AI! ⚡

## How to Contribute

### Reporting Issues

- Use [GitHub Issues](https://github.com/bk-hash5/the-ark/issues) to report bugs or request features
- Include steps to reproduce for bugs
- Include your Lightning wallet type if payment-related

### Suggesting Services

We're always looking to add useful AI services. If you have an idea:

1. Open an issue with the `service-request` label
2. Describe the service, expected input/output, and target audience
3. Suggest a fair price in sats

### Code Contributions

1. Fork the repository
2. Create a feature branch (`git checkout -b feat/your-feature`)
3. Make your changes
4. Write clear commit messages using [Conventional Commits](https://www.conventionalcommits.org/):
   - `feat:` new features
   - `fix:` bug fixes
   - `docs:` documentation changes
   - `security:` security improvements
   - `refactor:` code refactoring
5. Push to your fork and open a Pull Request

### Commit Message Format

```
<type>: <short description>

<optional body with more details>
```

Examples:
```
feat: add terraform generation service
fix: prevent duplicate image generation on concurrent polls
docs: update README with correct service count
security: add append-only audit logging
```

## Code of Conduct

- Be respectful and constructive
- Focus on the work, not the person
- Welcome newcomers — we all started somewhere

## Security

If you discover a security vulnerability, please **do not** open a public issue. Instead, email the details to the operator. Responsible disclosure is appreciated.
